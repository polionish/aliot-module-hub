Telegram Bot для мониторинга и уведомлений

Этот модуль предоставляет обновления в реальном времени о состоянии вашего умного дома через Telegram. Используя Telegram Bot API и Yandex API, бот периодически проверяет состояние устройств и отправляет персонализированные уведомления на основе заданных условий.

Возможности

  • Мониторинг состояния устройств в реальном времени.
  • Настраиваемые уведомления с использованием естественного языка.
  • Интеграция с Google Calendar для уведомлений по событиям.

Установка

  1. Добавьте бота в Telegram через уникальную ссылку.
  2. Настройте доступ к Yandex API в настройках бота.
  3. Задайте условия уведомлений с помощью команд.

Telegram Bot для шаринга сценариев

Этот модуль упрощает выполнение сценариев умного дома для внешних пользователей. Он позволяет генерировать уникальные ссылки для предустановленных сценариев, которые можно безопасно делиться с гостями без предоставления прямого доступа к системе.

Возможности

  • Создание ссылок для сценариев умного дома.
  • Удобное управление для гостей.

Установка

  1. Установите бота через Telegram.
  2. Создайте сценарий в приложении Яндекса.
  3. Сгенерируйте ссылку с помощью команды бота.

Расширение для браузера (YAML)

Этот модуль позволяет управлять сценариями на основе YAML прямо из браузера. Оно интегрируется с веб-интерфейсом Yandex IoT, обеспечивая синхронизацию конфигураций между локальными папками и облаком Яндекса.

Возможности

  • Импорт и экспорт YAML-сценариев.
  • Синхронизация с облаком Яндекса.

Установка

  1. Установите расширение из магазина браузера.
  2. Авторизуйтесь в Yandex IoT.
  3. Загрузите YAML-файл через интерфейс расширения.

Плагин для Obsidian

Этот плагин превращает Obsidian в среду разработки для редактирования YAML-сценариев. Он предлагает подсветку синтаксиса и контроль версий, что идеально для продвинутых пользователей.

Возможности

  • Подсветка синтаксиса YAML.
  • Контроль версий сценариев.

Установка

  1. Скачайте плагин из репозитория Obsidian.
  2. Добавьте его в настройки Obsidian.
  3. Начните редактировать YAML-файлы.
